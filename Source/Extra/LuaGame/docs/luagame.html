LuaGame Reference Manual

Version 0.1.0

Usage

Setup: 
using namespace Pxf::Extra::LuaGame;
Game *luagame = new Game("test.lua", m_Device);
luagames->Load();
m_Device is a pointer to your current Pxf::Graphics::Device instance.

Once every frame: 
luagame->Update(delta);
luagame->Render();
delta is a float delta value in seconds since last frame.
